1. To init project run 
```bash
make init
```

2. To download model
```bash
make prepare_artifacts
```

3. To export model to ONNX 
```bash
make export_model_to_onnx
```

4. To start app
```bash
make start
```

5. To build docker
```bash
make build_docker
```

6. To start app with docker
```bash
make start_docker
```

For other utilities navigate to [**Makefile**](Makefile)
